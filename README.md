### Estrutura de Pastas:

```text
projeto/
│
├── app.py                  # Código fonte do programa
├── requirements.txt        # Arquivo de dependências
│
├── default/                # OBRIGATÓRIA
│   └── modelo.docx         # O template do Word (apenas 1 arquivo aqui - obrigatório "__TEXTO__")
│
└── resultado-pdf/          # Criada automaticamente para salvar os PDFs

```
### Dependências:

```bash
pip install -r requirements.txt

```
